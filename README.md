# Gmail Intelligence Platform

## Overview
AI-powered Gmail management platform built using React, FastAPI and PostgreSQL.

## Features
- Gmail Integration
- Email Dashboard
- Email Categorization
- Email Summarization
- AI Assistant
- Compose Email
- Thread Awareness
- Analytics Dashboard

## Tech Stack
- React.js
- FastAPI
- PostgreSQL
- Gmail API
- Recharts

## Setup

### Backend
cd backend

pip install -r requirements.txt

uvicorn main:app --reload

### Frontend
cd frontend

npm install

npm run dev

## Architecture

Frontend (React)
↓
FastAPI Backend
↓
PostgreSQL
↓
Gmail API
