# Gmail Intelligence Platform

## Overview
AI-powered Gmail management platform built using React, FastAPI and PostgreSQL.

## Features
- Gmail Integration
- Email Dashboard
- Email Categorization
- Email Summarization
- AI Assistant
- Compose Email
- Thread Awareness
- Analytics Dashboard

## Tech Stack
- React.js
- FastAPI
- PostgreSQL
- Gmail API
- Recharts

## Run Project

### Backend
uvicorn main:app --reload

### Frontend
npm install
npm run dev
